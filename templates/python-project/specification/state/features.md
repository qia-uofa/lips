# Blackhole Simulation Features

## Core Functionalities
- **Gravitational Field Calculation**: Models Newtonian and relativistic gravitational effects
- **Particle Trajectory Simulation**: Computes paths of particles near the event horizon
- **Event Horizon Visualization**: Renders the Schwarzschild radius
- **Time Dilation Effects**: Simulates relativistic time dilation near the blackhole
- **Data Export**: Outputs simulation results in CSV and JSON formats

## User Workflows
1. **Basic Simulation**:
   - User provides blackhole mass and particle count
   - System simulates particle trajectories
   - Outputs visualization and data files

2. **Advanced Analysis**:
   - User configures simulation parameters (mass, spin, charge)
   - System runs extended simulation
   - Generates detailed reports on particle behavior

## Known Limitations
- Currently uses simplified physics models (not full general relativity)
- Limited to non-rotating (Schwarzschild) blackholes
- Particle interactions are approximated

## Future Improvements
- Add Kerr blackhole (rotating) support
- Implement more accurate relativistic calculations
- Add 3D visualization capabilities
